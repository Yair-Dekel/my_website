Portfolio

Chess

In this project I created Chess game based on the concept of Multi-Agent System (MAS). The main idea is to simulate a game which every piece on the board is an individual agent, in difference from regular game that played by two players. I created a game of two rooks and king vs king, and game of king and pawn vs king.

Emergency Buttom

I worked on this project with a team as part of a Software Engineering course. The project integrated both backend and frontend development. We created a database of community users and built an application that allows users to post tasks when they need help, and for volunteers to respond and assist. The app also includes an emergency button that users can activate to immediately notify others that they need urgent help.

Genetic Algorithm (GA)

Genetic Algorithms simulate the process of evolution and use principles from biology and natural selection to solve problems that are hard to handle with traditional algorithms (like NP-Complete problems). In this project, I built a GA engine to solve the Sudoku problem and the Bin Packing problem.
